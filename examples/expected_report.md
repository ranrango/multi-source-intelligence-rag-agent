# 多源情报分析报告示例

## 核心结论

当前样例数据会输出中高风险研判。报告会引用公开新闻、社交媒体、雷达日志、内部维护通知和历史事件简报，并说明“计划巡检”和“计划外悬停”之间的区别。

## 期望检查点

- 至少召回 3 条证据。
- 报告中出现证据编号，例如 `[sensor_log_radar-1]`。
- 输出风险等级、风险分数、关键证据、时间线、实体识别和不确定性。
- `logs/agent_trace.jsonl` 在运行后记录检索和工具调用轨迹。

## 演示命令

```bash
python3 -m src.app.cli --question "北岭港东侧堤坝无人机活动是否异常？请说明证据和风险。"
python3 scripts/smoke_test.py
```
